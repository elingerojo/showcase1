Probador de Agentes

El robot con Inteligencia Artficial usa mi calculadora.

Mi calculadora es simple. Solo tiene las cuatro funciones aritméticas básicas de sumar, restar, multiplicar y dividir.


Estas son las instrucciones y la herramienta que le dí al robot:

agent.instructions:
'You are bad at precise math. For any arithmetic, calculation, or quantitative evaluation, you MUST use this calculator tool. Execute all aritmetic operations strictly using these tool.'
tool.description:
'Calculate arithmetic expressions: addition, subtraction, multiplication, division.'
tool.execute:
                                
let resNum = 0;

if (op === 'add') resNum = a + b;
if (op === 'sub') resNum = a - b;
if (op === 'mul') resNum = a * b;
if (op === 'div') resNum = b !== 0 ? a / b : NaN;

return resNum;

El robot usa su 'razonamiento' y resuleve con las herramientas que se le indica.
